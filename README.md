## 📥 Cara Download

1. Pilih versi yang ingin diunduh (folder `vX.X.X`)
2. Pilih file sesuai platform Anda:
   - **Windows**: File `.exe` (installer) atau `.zip` (portable)
   - **Linux**: File `.deb`, `.rpm`, atau `.tar.gz`
   - **macOS**: File `.dmg` (installer) atau `.zip` (portable)
3. Verifikasi integritas dengan file `checksums.txt`
