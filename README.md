## ✨ Fitur Utama

| Fitur | Keterangan |
|-------|-----------|
| 🗄️ **Multi-Database** | SQLite (standalone), MySQL, PostgreSQL (client-server) |
| 🔐 **Keamanan** | HTTPS, audit trail komprehensif, RBAC, lisensi ECDSA offline |
| 📄 **Manajemen Dokumen** | CRUD, pencarian global, PDF/Excel export, template dinamis |
| 📊 **Dashboard Analitik** | Grafik interaktif, statistik real-time, aktivitas terbaru |
| 📋 **Laporan Agregat** | PDF profesional, analisis operator, distribusi barang hilang |
| 🔄 **Backup & Migrasi** | Hot-backup SQLite, streaming migrasi ke MySQL/PostgreSQL |
| 🖥️ **Mode Kiosk** | Tampilan TV lobi dengan 4 tema, QR code, info publik |
